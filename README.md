# 📱 乐吧公益中心 Android App

乐吧四端主链的Android客户端。

## 这是什么？

一个WebView壳App，加载乐吧GP入口页，在App内访问四端系统：
- 🏢 **机构管理端** — 入园签退·实时看板
- ⚡ **晚托CRM系统** — 学生管理·作业·评估
- 👪 **家长服务端** — 查看孩子在校状态
- 🎓 **课程报名** — 公益课程在线报名

## 自动构建

每次push到main分支，GitHub Actions自动构建APK并发布到Release。

## 安装

1. 去 [Releases](https://github.com/mars1417/leba-android-app/releases) 下载最新 APK
2. 手机打开APK文件 → 安装
3. 首次打开按提示允许通知权限

> 小米/华为用户：安装后建议进「设置→应用→自启动」开启自启动
